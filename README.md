# 🛠️사용 기술 및 라이브러리

- HTML/CSS
- React Native
- TypeScript
- Zustand
- Supabase

# 🐣 담당한 기능(현재 완료된 기능)

- 회원 가입 및 로그인
- 챌린지 추가 및 포기하기
- 지출 기록하기
- 챌린지 별 지출 목록 보기

# 🔥 추후 개발 예정인 기능

- 커뮤니티 기능(자랑하기)
- 가계부 작성 시 포인트 지급 기능(Google Admob)
